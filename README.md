# Healthcare IT Ticketing System

A full-stack ASP.NET Core MVC application designed to simulate an enterprise healthcare IT support platform.

The application allows users to create, manage, search, and analyze healthcare support tickets while providing operational dashboards and business intelligence reporting.

---

## Features

- Create, edit, delete, and view support tickets
- Dashboard with operational KPIs
- Search and filter tickets
- Priority and status badges
- Ticket Details page
- Reports Dashboard
- Interactive Chart.js visualizations
- Responsive Bootstrap interface

---

## Built With

- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- Bootstrap 5
- Chart.js
- LINQ
